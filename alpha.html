In this page, we will first explore nuclear physics through one of its simplest processes: Alpha Decay.

Alpha decay is an important process used to stabilise unstable radioactive isotopes (radioactive isotopes are atoms of the same element but each has a different number of neutrons, while having the same number of protons and electrons). It works by emitting an alpha particle (which is a helium nucleus consisting of 2 protons and 2 neutrons) from the nucleus.

This process reduces the size and mass of large, unstable nuclei, allowing them to reach a more stable state with lower energy.

You should be able to see that an alpha particle, consisting of two neutrons and two protons, is emitted during alpha decay. This reduces the mass and size of the large unstable nucleus, helping it reach a more stable, lower-energy state.

Key Notes:

  • The alpha particle is the same as a helium nucleus (2 protons & 2 neutrons).
  • It helps increase nuclear stability.
  • It is energetically favourable because ejecting this “package” releases energy.
  • Alpha decay usually occurs in elements with atomic number greater than 82.
